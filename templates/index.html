Welcome to the Friendly Task

This web app is meant to help people who are struggling with executive function, by providing the tools and the community that can help and support at the same place.

About the Project

Friendly Task is a web application designed to help users manage their daily tasks in a simple and efficient way. The project was created to provide an easy-to-use platform where users can add, edit, and complete tasks while also getting useful tips and advice from the community. Particularly, I would like to point out r/ADHD as my inspiration, but with wider functionality.

Features Implemented:

  • User authentication: Users can register, log in, and log out of the application.
  • Task management: Users can add, edit, start, and complete tasks.
  • Time management: The application helps users to prevent overloading their day with tasks exceeding 8 hours.
  • Tips and advice: Users can share, comment on, and like tips from the community.
  • Profile settings: Users can customize their profile by changing their username, uploading a profile picture, and toggling between light and dark mode.

Planned Features for Future Implementation:

  • Calendar integration: Users will be able to view their tasks in a calendar format and sync with popular calendar applications.
  • Task reminders and notifications: Users will receive reminders and notifications for upcoming tasks and deadlines.
  • Tagging and categorizing tasks: Users will be able to organize tasks using tags and categories for more efficient management.
  • Search functionality: Users will be able to search for tasks and tips using keywords and filters.
  • Social features: Users will be able to follow others and view tasks and tips from their connections.
  • Reward system and shop: Users can already earn tokens by completing tasks and participating in the community. Tokens will be used to purchase gifts and items from the shop.
  • Presents and popularity: Users can send presents to other users, increasing the popularity of their tips. The more tokens spent on a tip, the higher its visibility in the community.

FAQs

Q: How do I register for Friendly Task?
A: Visit the "Register" page, fill in the required information, and click the "Register" button.
Q: How do I add a new task?
A: Navigate to the "My Day" page and fill out the form with the task title and duration, then click "Add Task."
Q: Can I edit or delete a task?
A: Currently, you can only mark a task as started or completed. Editing and deleting tasks will be available in future updates.
Q: What is the maximum task duration allowed in a day?
A: You can't assign more then 8h of tasks a time, as it would not promote healthy productivity, but rather a toxic competition, which can't benefit anyone.
Q: How does the reward system work?
A: Users get the reward by completed the task that were set by them, however, we decided to postpone the introduction of the token economy, as for now the reward system has vulnerabilities
Q: SHOP??? Lootboxes??? Again?
A: No, there will be no monetization of my project, shop is more like a rank in CS:GO or Dota 2, the better you play the higher your rank. In our case it will not the ranks, but the things you will be able to buy and that will show up in your profile.
